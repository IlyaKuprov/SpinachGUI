gTrackBar is now built as a normal .NET Framework 4.8 class library by current Visual Studio.

Build it with:

  msbuild gTrackBar.sln /p:Configuration=Release /p:Platform="Any CPU"

The project's AfterBuild target copies gTrackBar.dll into the GUI directory so the
C++/CLI project can consume it with:

  #using "gTrackBar.dll"

The old netmodule workflow is no longer used.
